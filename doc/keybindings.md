# Neovim 快捷键参考

## 自定义快捷键

### Git
| 快捷键 | 作用 |
|--------|------|
| `<leader>gb` | 打开左侧 Git Blame 列（每行显示作者/日期） |
| `<leader>gB` | 弹窗显示当前行完整 Blame 信息 |

### 编辑
| 快捷键 | 作用 |
|--------|------|
| `<leader>ci` | 注释/取消注释当前行或选中块 |
| `gx` | 用浏览器打开光标下的 URL 或模块路径 |

### 搜索
| 快捷键 | 作用 |
|--------|------|
| `<leader>sp` | 用剪贴板内容全局搜索（自动转义特殊字符） |

## LazyVim 内置快捷键

### 文件/搜索
| 快捷键 | 作用 |
|--------|------|
| `<leader><space>` | 查找文件（Root Dir） |
| `<leader>ff` | 查找文件 |
| `<leader>fg` | 全局搜索（Grep） |
| `<leader>/` | 全局 Grep |
| `<leader>fb` | 切换 Buffer |
| `<leader>fr` | 最近打开的文件 |

### 代码/LSP
| 快捷键 | 作用 |
|--------|------|
| `gd` | 跳转到定义 |
| `gr` | 查看引用 |
| `gi` | 跳转到实现 |
| `K` | 查看文档悬浮窗 |
| `<leader>ca` | Code Action |
| `<leader>cr` | 重命名 |
| `<leader>cf` | 格式化 |
| `<leader>cd` | 查看当前行诊断 |
| `[d` / `]d` | 跳转到上/下一个诊断 |

### Git (Gitsigns)
| 快捷键 | 作用 |
|--------|------|
| `]h` / `[h` | 跳转到下/上一个 hunk |
| `<leader>ghs` | Stage hunk |
| `<leader>ghr` | Reset hunk |
| `<leader>ghp` | 预览 hunk |

### 窗口
| 快捷键 | 作用 |
|--------|------|
| `<leader>-` | 横向分割窗口 |
| `<leader>\|` | 纵向分割窗口 |
| `<C-h/j/k/l>` | 在窗口间移动 |
| `<C-Up/Down/Left/Right>` | 调整窗口大小 |

### Buffer
| 快捷键 | 作用 |
|--------|------|
| `<S-h>` / `<S-l>` | 切换到上/下一个 Buffer |
| `<leader>bd` | 关闭当前 Buffer |
| `<leader>bo` | 关闭其他 Buffer |

## Vim 原生快捷键

### 移动
| 快捷键 | 作用 |
|--------|------|
| `gg` / `G` | 跳到文件开头/结尾 |
| `{` / `}` | 跳到上/下一个空行（段落） |
| `%` | 跳转到匹配的括号 |
| `*` / `#` | 搜索光标下的单词（前/后） |
| `Ctrl+o` / `Ctrl+i` | 跳回/跳前（历史位置） |
| `H` / `M` / `L` | 跳到屏幕顶部/中部/底部 |
| `zz` | 将当前行滚动到屏幕中央 |

### 编辑
| 快捷键 | 作用 |
|--------|------|
| `ciw` | 修改当前单词 |
| `ci"` / `ci(` | 修改引号/括号内的内容 |
| `di"` / `di(` | 删除引号/括号内的内容 |
| `yiw` | 复制当前单词 |
| `dd` | 删除当前行 |
| `yy` | 复制当前行 |
| `p` / `P` | 粘贴到光标后/前 |
| `u` / `Ctrl+r` | 撤销/重做 |
| `>>` / `<<` | 缩进/反缩进 |
| `J` | 将下一行合并到当前行 |
| `~` | 切换大小写 |

### 折叠
| 快捷键 | 作用 |
|--------|------|
| `za` | 折叠/展开当前块 |
| `zc` / `zo` | 折叠/展开当前块 |
| `zM` | 折叠所有 |
| `zR` | 展开所有 |

### 选择
| 快捷键 | 作用 |
|--------|------|
| `v` | 进入字符选择模式 |
| `V` | 进入行选择模式 |
| `Ctrl+v` | 进入块选择模式 |
| `viw` | 选中当前单词 |
| `vi"` / `vi(` | 选中引号/括号内容 |
| `vip` | 选中当前段落 |

### 搜索替换
| 快捷键 | 作用 |
|--------|------|
| `/` | 向下搜索 |
| `?` | 向上搜索 |
| `n` / `N` | 跳到下/上一个匹配 |
| `:%s/old/new/g` | 全文替换 |
| `:s/old/new/g` | 当前行替换 |
